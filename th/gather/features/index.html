      คุณสมบัติฟอรัมชุมชน

      พึ่งพาอาศัย

      Zendesk Gather ช่วยให้คุณสร้างชุมชนสำหรับลูกค้าของคุณในการเชื่อมต่อและทำงานร่วมกัน Gather จะช่วยให้คุณสามารถให้การสนับสนุนได้ทุกเมื่อที่พวกเขาต้องการ

      สำหรับสมาชิก

      ช่วยให้พวกเขาช่วยเหลือกันเอง

      จัดเตรียมพื้นที่ที่สมาชิกสามารถมีปฏิสัมพันธ์กัน ค้นหาคำตอบของคำถาม และแลกเปลี่ยนฟีดแบ็ค

      ค้นหาข้อมูลที่เหมาะสม

      การสืบค้นที่ทรงพลัง

      กรองการสืบค้นภายในศูนย์ความช่วยเหลือเพื่อค้นหาโพสต์ในชุมชนและกรองผลลัพธ์ตามหัวข้ออย่างง่ายดาย

      การนาวิเกตที่ราบรื่น

      สร้างความมั่นใจว่าสมาชิกสามารถค้นหาความช่วยเหลือที่ต้องการได้โดยจัดระเบียบชุมชนไว้เป็นหัวข้อที่สามารถเรียกดูได้

      ส่งเสริมเนื้อหาเด่น

      นำเสนอเนื้อหาที่ตรงประเด็นโดยปักหมุดโพสต์ในชุมชนของคุณ

      การโหวต

      ผลักดันเนื้อหายอดนิยมและถูกต้องด้วยการโหวตเพิ่มและโหวตลดอันดับโพสต์และข้อคิดเห็น

      โพสต์ที่คล้ายกัน

      แนะนำโพสต์ที่คล้ายกันเพื่อหลีกเลี่ยงการตั้งเธรดซ้ำและตอบคำถามด้วยเนื้อหาที่มีอยู่แล้ว

      มีส่วนร่วม

      ตั้งเธรดใหม่

      ดำเนินการในชุมชนโดยใช้เครื่องมือแก้ไขแบบ WYSIWYG เพื่อจัดรูปแบบโพสต์ ลากแล้ววางภาพ หรือแทรกลิงก์ได้อย่างง่ายดาย

      โปรไฟล์ผู้ใช้

      จัดการกิจกรรมและสถานะของชุมชนได้ในที่เดียวด้วยตัวเลือกในการสร้างนามแฝง

      ติดตาม

      รับการแจ้งเตือนทางอีเมลด้วยการติดตามหัวข้อ โพสต์ หรือสมาชิกคนอื่นๆ

      อ้างถึง

      แท็กสมาชิกในเธรดเพื่อเริ่มการสนทนาเพื่อการมีส่วนร่วมและการเติบโตที่ดียิ่งขึ้น

      การแชร์ทางโซเชียล

      แชร์เนื้อหาชุมชนใน Facebook, Twitter หรือ LinkedIn

      สำหรับผู้ดูแล

      ดำเนินการเบื้องหลัง

      จัดระเบียบเนื้อหาอย่างมืออาชีพเพื่อให้คุณสามารถจัดการและขยายชุมชนของคุณด้วยทรัพยากรที่เหมาะสมเพื่อเพิ่มการมีส่วนร่วมของสมาชิก

      ไม่พลาดทุกปัญหา

      ส่งต่อโพสต์

      ส่งต่อโพสต์ไให้เจ้าหน้าที่สนับสนุนของคุณเมื่อจำเป็นต้องให้ความช่วยเหลือแบบตัวต่อตัว

      สถานะของโพสต์

      แท็กสถานะปัจจุบันของโพสต์ ทั้งเมื่อมีการตอบคำถาม และเมื่อมีการวางแผนคำขอผลิตภัณฑ์ เมือไม่ได้วางแผนไว้ หรือเมื่อเสร็จสมบูรณ์แล้ว

      ผู้ดำเนินการดูแลชุมชน

      ให้อำนาจสมาชิกในการเป็นผู้ดูแลโดยการกำหนดระดับการอนุญาตที่แตกต่างกัน เช่นความสามารถในการกำหนดสถานะ ซ่อน แก้ไข และย้ายโพสต์

      ปรับขยายได้ง่ายดาย

      การควบคุมเนื้อหา

      ใช้ตัวกรองคีย์เวิร์ดเพื่อตั้งค่าสถานะโพสต์ทันทีเพื่อตรวจสอบและอนุมัติก่อนที่จะเผยแพร่

      บล็อกสแปม

      กำหนดสถานะเนื้อหาที่น่าสงสัยโดยอัตโนมัติเพื่อตรวจสอบและอนุมัติก่อนเผยแพร่

      แดชบอร์ดชุมชน

      ติดตามเมตริกที่สำคัญเช่นโพสต์ ข้อคิดเห็น จำนวนครั้งที่ดู จำนวนโหวต การสมัครสมาชิก และคำตอบด้วยตัวกรองตามแบรนด์และช่วงวันที่ได้อย่างง่ายดาย

      การวิเคราะห์สถิติ Google

      ผู้จัดการชุมชนสามารถติดตามเมตริกที่เกี่ยวข้อง เช่นจำนวนครั้งที่ดูหน้าเว็บ การอ้างอิง และอัตราย้อนกลับ

      ทำให้เป็นของคุณเอง

      ธีมที่ปรับเปลี่ยนเองได้

      สร้างแบรนด์ของชุมชนอย่างง่ายดายด้วยธีมที่มีให้อยู่แล้วหรือใช้ HTML, CSS และ Javascript

      หลายภาษา

      แปลชุมชนของคุณเป็นมากกว่า 40 ภาษา เพื่อให้คุณสามารถบริการสมาชิกได้ในภาษาท้องถิ่นของพวกเขา

      ควบคุุมการเข้าถึง

      เลือกผู้ที่สามารถเข้าถึงชุมชนของคุณด้วยตัวเลือกเพื่อเรียกขอการลงทะเบียนหรือทำให้เป็นสาธารณะ

      API เพื่อปรับตั้งค่า

      ออกแบบการปรับแต่งที่ซับซ้อนเพื่อส่งเสริมประสบการณ์สำหรับลูกค้า

      หลายแบรนด์

      สร้างชุมชนได้หลายแห่งเพื่อให้บริการกลุ่มลูกค้า ภูมิภาค หรือแบรนด์ที่หลากหลาย แต่ละชุดจะแยกต่างหากเพื่อให้คุณสามารถกำหนดการให้อนุญาตที่แตกต่างกัน สร้าง URL ที่ไม่เหมือนกัน และปรับแต่งธีมต่าง ๆ ให้เข้ากัน

      พร้อมสำหรับเซิร์ชเอนจิน

      Gather พัฒนาอย่างเต็มประสิทธิภาพเพื่อให้เซิร์ชเอนจินสามารถเข้าใจเนื้อหาของคุณได้ และสมาชิกสามารถค้นพบคำตอบไม่ว่าพวกเขาจะหาคำตอบจากที่ไหน
