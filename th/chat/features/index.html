เริ่มต้น
การสนทนา

เข้าถึงลูกค้าของคุณได้ทันทีผ่านเว็บ มือถือ และการส่งข้อความ
ดูวิดีโอ
ภาพรวม คุณลักษณะเด่น ราคา ติดต่อเรา
เริ่มทดลองใช้ฟรี
  • ทดลองใช้ Chat ได้ฟรี

    การแชทเดียวมีค่าเทียบเท่าอีเมลพันฉบับ

    เพิ่มวิดเจ็ตแชทลงไปยังเว็บไซต์หรือเชื่อมต่อกับช่องทางการส่งข้อความของคุณและเริ่มพูดคุยกับลูกค้าได้ในเวลาไม่กี่นาที ซึ่งการแชทจะช่วยให้คุณสามารถช่วยเหลือลูกค้าได้แบบเรียลไทม์ ซึ่งเพิ่มความพึงพอใจของลูกค้าได้ และลูกค้าที่ถึงพอใจคือลูกค้าที่มีความสุขมากขึ้น

    ชมการสาธิตการใช้งาน

    สร้างความมีส่วนร่วม

    พูดคุยกัน
    ผ่านปลายนิ้ว

    ลูกค้ามีแนวโน้มเพิ่มขึ้นสามเท่าที่จะสั่งซื้อเมื่อคุณเริ่มการแชทกับพวกเขาแบบเชิงรุก ไม่ว่าคุณจะอยู่ใน Support หรือ Sales ก็สามารถใช้ Chat เพื่อติดตามลูกค้าและติดต่อกับพวกเขาในเวลาที่เหมาะสมได้

    ทริกเกอร์

    ส่งข้อความที่มีเป้าหมายและอิงพฤติกรรมไปยังลูกค้า ตัวอย่างเช่น คุณสามารถช่วยให้ลูกค้าทำการสั่งซื้อให้เสร็จสมบูรณ์โดยการติดต่อพวกเขาทางแชทโดยอัตโนมัติ

    แบบฟอร์มก่อนการแชทและแบบฟอร์มการออฟไลน์

    ใช้แบบฟอร์มก่อนการแชทเพื่อขอข้อมูลการติดต่อจากผู้เยี่ยมชมก่อนที่จะเริ่มการแชท หรือแบบฟอร์มการออฟไลน์เพื่อให้แน่ใจว่าลูกค้าสามารถติดต่อคุณได้ แม้ว่าคุณจะไม่อยู่

    รายชื่อผู้เยี่ยมชม

    บริษัทให้เช่าเหมาเรือตกปลาอย่าง Fishfishme ใช้การแชทในการติดตามตรวจสอบผู้เยี่ยมชมเว็บไซต์ของบริษัทและหน้าเพจที่พวกเขาเรียกดู วิธีนี้ช่วยให้พวกเขาพุ่งเป้าไปยังผู้ใช้แต่ละรายและเสนอความช่วยเหลือเมื่อพวกเขาต้องการได้

    การให้คะแนนการแชท

    Chat มอบการให้คะแนนความพึงพอใจสูงสุดในบรรดาช่องทางการสนับสนุนทั้งหมด แต่ก็ยังสามารถพัฒนาให้ดีขึ้นได้ ใช้การให้คะแนนการแชทเพื่อรวบรวมฟีดแบ็คและพัฒนาประสิทธิภาพของคุณต่อไป

    การส่งไฟล์

    "ภาพภาพเดียวมีค่าเท่าคำพูดนับพัน" ประโยคที่ฟังดูเกร่อ แต่เป็นประโยคอมตะสำหรับการสนับสนุนลูกค้า แชร์ไฟล์ใดก็ได้กับลูกค้าของคุณ ไม่ว่าจะเป็นภาพถ่ายหน้าจอ คู่มือผลิตภัณฑ์ หรือไฟล์ภาพเคลื่อนไหว GIF จะช่วยแก้ปัญหาได้เร็วขึ้น

    แชทบอท

    ผนวกรวมบอทที่ขับเคลื่อนด้วยปัญญาประดิษฐ์ลงใน Chat โดยใช้ API ของเรา อ่านรายละเอียดเกี่ยวกับวิธีการทำงานที่ง่ายแต่กินเวลาให้เป็นแบบอัตโนมัติ และให้ความช่วยเหลือแก่เจ้าหน้าที่ของคุณ

    วิเคราะห์

    ดูวิธีการวัดข้อมูล

    เห็นภาพรวมของแชทและกิจกรรมเจ้าหน้าที่ของคุณเพื่อให้คุณมีข้อมูลในการถามคำถามที่ดี

    การรายงานแชทและเจ้าหน้าที่:

    ติดตามระดับการแชทของคุณ ตรวจสอบประสิทธิผลเจ้าหน้าที่ และวิเคราะห์เมตริกเพื่อคาดการณ์ความต้องการของลูกค้าและเพิ่มความพึงพอใจ

    การติดตามตรวจสอบแบบเรียลไทม์:

    ติดตามตรวจสอบข้อมูลสดเกี่ยวกับปริมาณการแชท ประสบการผู้เยี่ยมชม และประสิทธิภาพเจ้าหน้าที่

    ติดตามการแปลงโอกาสเป็นการขาย:

    สร้างเป้าหมายทางธุรกิจเพื่อประเมินว่าลูกค้าของคุณสามารถบรรลุเป้าหมายต่างๆ อย่างการสั่งซื้อหรือลงทะเบียนจดหมายข่าวได้บ่อยเพียงใด


    การวิเคราะห์สถิติเพิ่มเติม

    ปรับแต่ง

    ความช่วยเหลือที่ดูดี

    วิดเจ็ตแชทเป็นจุดสัมผัสของแบรนด์ที่สำคัญสำหรับลูกค้า ทำให้แน่ใจว่าจุดนี้สะท้อนถึงแบรนด์ของคุณ เพื่อให้ลูกค้ารู้ว่าพวกเขากำลังพูดคุยกับแบรนด์อะไร

    ช่องการแชท
    ช่องการแชทที่ปรับแต่งได้ของเราช่วยให้คุณสร้างภาพลักษณ์ที่เชิญชวนให้แก่วิดเจ็ตแชทได้ เลือกเลย์เอาท์และธีมที่หลากหลาย หรือจะเปลี่ยนภาพใหม่ทั้งหมดก็ย่อมได้ ทางเลือกเป็นของคุณ
    Web SDK
    สร้างแชทวิดเจ็ตฝั่งผู้ใช้ปลายทางแบบกำหนดเองทั้งหมดให้ผู้เยี่ยมชมได้รับประสบการณ์เฉพาะตัวที่มีความลื่นไหลต่อเนื่องกัน
    บริการบุคคลที่สามต่าง ๆ
    สามารถเพิ่ม Chat ลงใน CMS ใดก็ได้ เช่น WordPress, Drupal, Joomla, Wix, Squarespace, Shopify และ PrestaShop.

    การปรับแต่งอื่นๆ

    จัดระเบียบ

    พัฒนาตามการเติบโตของธุรกิจ

    การมอบการสนับสนุนไปพร้อมกับการทำให้ธุรกิจเติบโตอาจเป็นเรื่องยาก คุณสมบัติการจัดการเวิร์กโฟลว์ที่ช่ำชองของเราทำให้สิ่งต่าง ๆ ง่ายขึ้น

    ชมการสาธิตการใช้งาน

    การกำหนดเส้นทางแชท

    มอบหมายการแชทขาเข้าไปยังเจ้าหน้าที่ที่กำลังปฏิบัติงานโดยอัตโนมัติ เพื่อรับรองว่าลูกค้าของคุณจะได้รับการสนับสนุนทันทีและเจ้าหน้าที่ของคุณจะไม่รับภาระมากเกินไป

    แท็กแชทต่าง ๆ

    เพิ่มแท็กไปยังแชทของคุณ สร้างบริบทให้บทสนทนาของคุณ และเข้าใจลูกค้าของคุณได้ดียิ่งขึ้นห

    แผนกต่าง ๆ

    จัดระเบียบเจ้าหน้าที่ของคุณเข้าเป็นทีมต่าง ๆ โดยอ้างอิงจากทักษะ ภาษา ความชำนาญ หรือแม้แต่แบรนด์

    เวลาทำการ

    สร้างกำหนดการประจำวันหรือประจำสัปดาห์สำหรับวิดเจ็ตแชทและแสดงสถานะออนไลน์เฉพาะในเวลาทำการ

    ข้อความลัด

    สร้างการตอบกลับที่สร้างเท็มเพลตไว้ล่วงหน้าสำหรับคำร้องขอที่พบบ่อยและส่งมอบคำตอบที่รวดเร็วให้แก่ลูกค้า

    บทบาทและการให้อนุญาต

    ระบุสิทธิ์เฉพาะตัวให้กับเจ้าหน้าที่ และควบคุมว่าพวกเขาเข้าใช้อะไรใน Zendesk Chat ได้บ้าง

    ขยายทีมของคุณ

    สมาร์ทโฟน

    ทำงานได้ดีในทุกอุปกรณ์

    Mobile Chat SDK

    เพิ่มการแชทสดลงไปยังแอพอุปกรณ์พกพาของคุณโดยใช้
    SDK ที่เป็นมิตรต่อนักพัฒนาของเรา ด้วยโค้ดเพียงไม่กี่บรรทัด ลูกค้าของคุณสามารถขอความช่วยเหลือได้โดยไม่ต้องออกจากแอพของคุณ

    วิดเจ็ตสำหรับอุปกรณ์พกพา

    วิดเจ็ตแชทนี้เพิ่มประสิทธิภาพสูงสุดสำหรับหน้าจอทุกขนาดและทุกชนิด เพื่อให้แน่ใจว่าลูกค้าของคุณได้รับประสบการณ์ที่ลื่นไหลต่อเนื่องในทุกอุปกรณ์ของพวกเขา

    แอพสำหรับอุปกรณ์พกพา

    แอพพลิเคชันแชทสำหรับ Android และ iPhone ทำให้การสนับสนุนเป็นไปอย่างต่อเนื่องขณะที่เจ้าหน้าที่เดินทางไปที่ต่าง ๆ
