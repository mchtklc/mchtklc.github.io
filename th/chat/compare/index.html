    การรับรองความถูกต้องแบบสองปัจจัยคุณสมบัติการรักษาความปลอดภัยที่เลือกได้สำหรับผู้ดูแลและเจ้าหน้าที่ ทำให้ผู้ที่ไม่หวังดีลักลอบเข้าบัญชีของคุณได้ยากขึ้น ด้วยการกำหนดให้ใช้รหัสผ่านอุปกรณ์พกพา เพิ่มเติมจากรหัสผ่านตอนเข้าสู่ระบบ
    Single sign-on (SSO) for web and mobile**มอบการทำงานร่วมกันของ SSO กับระบบการจัดการตัวตนที่มีอยู่แล้วของคุณ (เช่น Active Director) SSO ของ Zendesk อาศัยเทคโนโลยีที่เรียกว่า JSON Web Token (JWT) หรือ Security Assertion Markup Language (SAML) สำหรับการรักษาความปลอดภัยในการแลกเปลี่ยนข้อมูลความถูกต้องของผู้ใช้
    		 JWT JWT / SAML JWT / SAML
    การแบนผู้เยี่ยมชมการแบนที่อยู่ IP ที่เจาะจงเพื่อหลีกเลี่ยงการสแปม
    การเรียกคืนข้อมูลเมื่อเกิดเหตุร้ายโปรแกรมการเรียกคืนข้อมูลเมื่อเกิดเหตุร้ายของเราทำให้มั่นใจว่าบริการของเราจะยังพร้อมให้บริการหรือเรียกคืนได้ง่ายในกรณีที่เกิดเหตุร้าย
    		มาตรฐาน มาตรฐาน มาตรฐาน เพิ่มสมรรถนะ (เมื่อคุณซื้อ Add-On การเรียกคืนข้อมูลเมื่อเกิดเหตุร้ายขั้นสูง)
    ประเทศที่ถูกบล็อก ซ่อนวิดเจ็ตแชท Zendesk Chat ในประเทศที่ระบุ
    โดเมนที่อนุญาต จำกัดเว็บไซต์ที่สามารถแสดงวิดเจ็ต Zendesk Chat ของคุณได้ และทำให้แน่ใจว่าไม่มีใครสามารถนำรหัสวิดเจ็ตแชทของคุณไปใช้ในโดเมนอื่นได้
    Credit card redactionAutomatically hide credit card numbers in chats, and remove it from your chat history and database to protect confidential information.
    ข้อจำกัดการเข้าถึง IPจำกัดการเข้าสู่ระบบหน้าแดชบอร์ดและบัญชี Zendesk Chat ผ่านที่อยู่ IP ที่ระบุหรือที่อยู่ในขอบเขต
    การรับรองอุตสาหกรรม (รวมทั้ง HIPAA)Zendesk ได้ผ่านการประเมิน HIPAA/HITECH อย่างเสร็จสมบูรณ์ และพร้อมให้บริการข้อตกลง Business Associate Agreement (BAA) เพื่อให้ผู้สมัครนำไปดำเนินการได้
    		จำเป็นต้องใช้บัญชี Zendesk Support

    ** Feature availability depends on Chat account type + requires Support
