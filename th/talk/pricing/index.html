              การใช้งาน

              การใช้งานประกอบด้วยหมายเลขโทรศัพท์ สรุปการสนทนา และตัวเลือกพิเศษการทำทรานสคริปต์ข้อความเสียง

              ปัจจัยหลายอย่างอาจส่งผลกระทบต่อค่าใช้จ่าย เช่น หากคุณให้หมายเลขโทรศัพท์ท้องถิ่นหรือหมายเลขโทรฟรี สนับสนุนฐานลูกค้าในประเทศหรือระหว่างประเทศ หรือโอนสายไปยังหมายเลขภายนอก

              คำนวณค่าใช้จ่ายในการใช้งาน

              หากคุณต้องการรายละเอียดเพิ่มเติม โปรดติดต่อเรา แล้วเราจะได้คุยกันอย่างละเอียด

              คำถามที่ถามกันบ่อย พร้อมคำตอบ

              การทดลองใช้

              การทดลองใช้ Zendesk Talk ทำงานอย่างไร

              การทดลองใช้ Zendesk Talk ของคุณเริ่มที่แผน Professional (นอกเสียจากคุณจะใช้แผน Essential ของ Zendesk Support เช่นนั้นการทดลองใช้ Talk ของคุณจะเริ่มที่แผน Team) ระหว่างการทดลองใช้ คุณจะมีหมายเลขโทรศัพท์ฟรี และมีเครดิตการโทร $1 ซึ่งจะครอบคลุมเวลาการโทรประมาณ 50 นาที การโทรออกทั้งหมดถูกจำกัดโดยอัตโนมัติที่สองนาทีและเฉพาะหมายเลขในท้องถิ่นหรือภายในประเทศเท่านั้น การโทรออกไปยังหมายเลขต่างประเทศระหว่างการทดลองใช้ของคุณ โปรดติดต่อฝ่ายสนับสนุน

              จะเกิดอะไรขึ้นเมื่อการทดลองใช้ Talk สิ้นสุดลง

              หากคุณเป็นลูกค้า Zendesk และต้องการจะใช้ Talk ต่อไปหลังช่วงการทดลองใช้ฟรี คุณจำเป็นต้องเพิ่ม Talk ลงไปในการสมัครใช้บริการ Zendesk ของคุณ หากคุณยังไม่ได้เป็นลูกค้า Zendesk คุณจำเป็นต้องซื้อการสมัครใช้บริการ Zendesk Support ก่อนใช้ Talk ต่อไป

              หมายเลข Talk จะเปิดใช้งานเมื่อคุณเริ่มต้นการทดลองใช้งาน Talk หรือ Suite และจะถูกเก็บไว้เมื่อคุณซื้อ Talk ในทุกแผน

              หากคุณไม่ต้องการหมายเลข Talk และ/หรือต้องการหลีกเลี่ยงค่าใช้จ่ายนี้ คุณจะต้องลบหมายเลขโทรศัพท์หลังการทดลองใช้งาน

              ฉันจะลองใช้ Text ได้อย่างไร

              Text เป็นคุณสมบัติหนึ่งของ Zendesk Talk การทดลองใช้ Zendesk Talk ฟรีนั้นรวมคุณสมบัติ Text ไว้แล้ว ค่าบริการ $1 และหมายเลขโทรศัพท์ฟรีใช้ได้กับทั้งการใช้งาน Talk และ Text

              หากคุณมีการทดลองใช้ Talk หรือบัญชีที่ชำระเงินแล้ว คุณอาจสามารถใช้หมายเลขที่มีอยู่แล้วสำหรับ Text ได้ ติดต่อฝ่ายสนับสนุนเพื่อรับข้อมูลเพิ่มเติม

              ซื้อ

              มี Zendesk Talk ให้ซื้อแยกจาก Zendesk Support หรือไม่

              ในการซื้อ Zendesk Talk คุณจำเป็นต้องซื้อการสมัครบริการ Zendesk Support ก่อน อ่านรายละเอียดเกี่ยวกับแผนดังกล่าว ได้ที่นี่

              ฉันจำเป็นต้องใข้ทั้ง Talk และ Text หรือไม่

              Talk Team และการสมัครใช้งานข้างต้นมีคุณสมบัติ Text รวมอยู่ด้วย หมายเลข Text ที่มีความสามารถในการโทรสามารถใช้กับ Talk ได้ แต่คุณก็สามารถเลือกเปิดใช้หมายเลขโทรศัพท์สำหรับ Text และ Talk แยกกันได้เช่นกัน

              ค่าใช้จ่ายคาดว่าจะเกิดขึ้น

              ค่าใช้จ่ายในการใช้งานของคุณจะขึ้นอยู่กับปริมาณการโทรและข้อความอักษรของทีม เช่นเดียวกับสภาพภูมิศาสตร์

              ค่าใช้จ่ายในการใช้งานหมายรวมถึงค่าหมายเลขโทรศัพท์ (คิดค่าบริการรายเดือน) การโทร (รวมขาเข้า ขาออกและข้อความเสียงคิดเป็นนาที) บริการเสริมบันทึกการโทร และการถอดข้อความเสียง (คิดเป็นนาที) และข้อความ (คิดตามจำนวนข้อความ) อัตรา SMS มาตรฐาน SMS คือ $0.0094 ต่อข้อความขาเข้า อัตราขาออกอาจแตกต่างไป ค้นหารายละเอียดเพิ่มเติมเกี่ยวกับราคาการใช้งาน Text ได้ ที่นี่

              การโทรจะคิดค่าบริการต่อนาที โดยปัดเศษไปเป็นจำนวนนาทีที่ใกล้เคียงที่สุด อัตราต่อนาทีจะคิดค่าบริการต่างกันไป ไม่ว่าจะเป็นการโทรเข้าออกในเบราว์เซอร์หรือจากหมายเลขภายนอก

              มีค่าธรรมเนียมการทำสัญญาหรือค่าใช้จ่ายเพิ่มเติมหรือไม่ ฉันสามารถเปลี่ยนแผนได้หรือไม่

              ไม่! คุณจ่ายให้กับเจ้าหน้าที่ และคุณจ่ายให้กับการใช้งานของคุณ เท่านั้นเอง! ไม่มีค่าธรรมเนียมการติดตั้งหรือเริ่มใช้งาน และลูกค้าเกือบทุกรายติดตั้งและเริ่มใช้อย่างสมบูรณ์ได้ภายในวันเดียว

              แผน Talk เป็นแบบเดือนต่อเดือนหรือรายปี คุณสามารถเริ่ม/หยุดหรือเปลี่ยนแปลงแผนของคุณได้ทุกเวลา (แต่โปรดทราบว่าไม่มีการคืนเงินสำหรับกรณีการยกเลิกหรือการดาวน์เกรด โปรดดู รายละเอียดในข้อกำหนดการใช้บริการ)

              ฉันต้องซื้อที่นั่ง Talk สำหรับเจ้าหน้าที่ทุกรายในบัญชี Support ของฉันหรือไม่

              ไม่ คุณไม่ต้องซื้อที่นั่ง Talk สำหรับเจ้าหน้าที่ทุกรายในบัญชี Support ของคุณ คุณสามารถเลือกได้ว่าเจ้าหน้าที่รายใดบ้างที่จะใช้ Talk

              ใช้งาน

              ฉันสามารถใช้หมายเลขโทรศัพท์ที่มีอยู่แล้วได้หรือไม่

              ในหลายๆ ประเทศ เราสามารถช่วยคุณย้ายหรือ "พอร์ต" หมายเลขโทรศัพท์ที่มีอยู่ไปยัง Zendesk Talk ได้ อ่านรายละเอียดเกี่ยวกับการพอร์ตหมายเลขโทรศัพท์ หรือคุณสามารถโอนสายไปยังหมายเลขการสนับสนุนที่มีอยู่ของคุณลงใน Talk ได้ คุณจะยังคงต้องตั้งค่าหมายเลข Talk จากนั้นทำงานร่วมกับผู้ให้บริการโทรศัพท์เดิมของคุณเพื่อโอนสายไปยังหมายเลข Zendesk Talk แทน อ่านรายละเอียดเกี่ยวกับการโอนสาย

              Talk มีพร้อมให้บริการในประเทศใดบ้าง

              Zendesk Talk ใช้ได้ทุกที่ที่คุณใช้ Zendesk สนับสนุนการโทรเข้าจากกว่า 70 ประเทศทั่วโลก อ่านรายละเอียดเกี่ยวกับความพร้อมใช้งานของหมายเลขโทรศัพท์ที่นี่
