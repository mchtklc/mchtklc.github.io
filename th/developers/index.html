Zendesk สำหรับนักพัฒนา

ต่อสู้เพื่อสิทธิ์ของคุณที่จะเขียนโปรแกรม

Zendesk เปิดกว้าง ยืดหยุ่น และสร้างขึ้นบน AWS คุณสามารถเชื่อมต่อแหล่งข้อมูลทั้งหมด สั่งสมประสบการณ์สำหรับลูกค้าที่เป็นนวัตกรรม และสร้างแอพหรือระบบอัตโนมัติใดๆ ที่คุณต้องการ

ดูแหล่งข้อมูล

ผู้ท้าชิงรายใหม่เข้าสู่สนาม

เครื่องมือสำหรับนักพัฒนาแบบเก่านั้นติดอยู่ในอดีต ล้าสมัย ไม่ยืดหยุ่น และขัดกับสามัญสำนึก และนั่นทำให้การสร้างแอพแห่งอนาคตเป็นเรื่องท้าทาย แต่เปล่า นี่ไม่ใช่ความท้าทายที่ดี ออกจะค่อนไปทางน่าหงุดหงิด น่ารำคาญ และไม่ฉลาดเสียมากกว่า

แพลตฟอร์มนักพัฒนา Zendesk ให้อิสระในการสร้างสิ่งที่คุณต้องการในแบบที่คุณต้องการ และเมื่อไม่มีอะไรมาฉุดรั้งคุณ เราขอเสนอความท้าทายที่ต่างออกไป: คุณจะสร้างอะไรได้บ้าง

เป็นมิตรกับนักพัฒนา อย่างแท้จริง

สร้างได้เร็วขึ้นและฉลาดยิ่งขึ้น

การเรียนรู้ภาษาการเขียนโปรแกรมหรือเฟรมเวิร์กใหม่ๆ ที่ใช้งานได้กับแค่แพลตฟอร์มเดียวนั้นไม่เหมาะสม Zendesk มีแนวคิดที่ดีกว่า นั่นก็คือสร้างและปรับใช้แอพได้เร็วขึ้นด้วยเครื่องมือสำหรับนักพัฒนาที่ใช้งานง่ายและใช้ไลบรารีโอเพ่นซอร์สที่คุณรู้จักและชื่นชอบ

อย่าปิดกั้นตัวเอง

อย่ายึดติดกับแพลตฟอร์มที่ไม่ยืดหยุ่นด้วย API ที่ตายตัว ซึ่งจำกัดวิธีและสถานที่ที่คุณสามารถสร้างได้ เข้าถึง รวบรวม และขยายข้อมูลของคุณได้ทุกที่ด้วยแพลตฟอร์มที่ขับเคลื่อนด้วยกิจกรรมและ API ที่มีประสิทธิภาพ ซึ่งช่วยให้คุณสร้างทุกอย่างได้ทั้งภายในและภายนอก Zendesk

ก้าวล้ำอยู่เสมอ

ไม่มีใครควรถูกกลั่นแกล้งด้วยเทคโนโลยี และไม่มีใครควรถูกบังคับให้ใช้ภาษาที่ล้าสมัยและมีกรรมสิทธิ์แล้ว ใช้ประโยชน์จากแนวโน้มล่าสุดและเครื่องมือที่ดีที่สุดเพื่อสร้างแอพและประสบการณ์ใหม่ๆ

เข้าใจแล้ว เอกสารอยู่ไหนล่ะ

ดูเอกสารประกอบ

สร้างทุกสิ่งที่คุณต้องการ

เชื่อมต่อข้อมูล

จัดเก็บและจัดการข้อมูลลูกค้าทั้งหมดของคุณผ่านโปรไฟล์ กิจกรรม วัตถุ และการสนทนาในแพลตฟอร์มเดียวด้วย Zendesk Sunshine

ทำให้ Zendesk Suite ดียิ่งขึ้น

ดูว่าคุณทำอะไรได้บ้างเพื่อเพื่อเพิ่มประสิทธิภาพให้ Zendesk Suite: สร้าง แอพที่สร้างเอง สำหรับ Support และ Chat ผสานรวมระบบโทรศัพท์ด้วย Talk CTI Toolkit และสร้าง ธีมที่สร้างเอง ด้วย API ศูนย์ความช่วยเหลือ

กระจายการสนับสนุนไปให้ทั่ว

ใช้ Zendesk Web Widget และ Mobile SDK เพื่อมอบความช่วยเหลือให้ถึงมือลูกค้าไม่ว่าจะอยู่ที่ใด ในเว็บไซต์ แอพสำหรับอุปกรณ์พกพา หรือแม้แต่ตู้เย็นอัจฉริยะของคุณ (ถ้าเกิดคุณอยู่ในธุรกิจตู้เย็นอัจฉริยะน่ะนะ)

ใส่การส่งข้อความไว้ทุกที่

รวมการสนทนากับลูกค้าและส่งมอบประสบการณ์การส่งข้อความแบบโต้ตอบผ่านเว็บ อุปกรณ์พกพา และช่องทางโซเชียลด้วย Sunshine Conversations

อย่าหยุดสตรีม

สตรีมข้อมูล Zendesk ของคุณ ลงในบริการ AWS หลักด้วย Amazon EventBridge หรือปรับปรุงเวิร์กโฟลว์และสร้างระบบอัตโนมัติใน Zendesk ด้วย API ได้เกือบทุกสิ่ง

อย่าเก็บไว้กับตัว

ส่งแอพและธีมของศูนย์ความช่วยเหลือที่สร้างขึ้นเองไปที่ Zendesk Marketplace ให้คนอื่นได้ใช้ด้วย
