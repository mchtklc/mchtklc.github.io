เปรียบเทียบแผนต่างๆ

คำถามที่ถามกันบ่อย พร้อมคำตอบ

การทดลองใช้

การทดลองใช้มีรายละเอียด อย่างไร

เมื่อคุณลงทะเบียนเพื่อทดลองใช้ Support คุณสามารถเข้าถึงคุณสมบัติระดับ Professional ทั้งหมด หากคุณต้องการทดลองใช้แผนใดโดยเฉพาะ โปรดติดต่อเรา  ในระหว่างการทดลองใช้ คุณสามารถเลือกแผนและชำระเงินด้วยบัตรเครดิตจากในบัญชีของคุณได้

ฉันทดลองใช้ Support, Chat, Guide และ Talk ได้หรือไม่

ได้สิ! คุณสามารถเริ่มการทดลองใช้งานทั้ง Guide, Chat และ Talk จากการทดลองใช้งาน Support ใหม่ของคุณ ลองใช้ทีละตัวหรือทั้งหมดพร้อม ๆ กันได้

เกิดอะไรขึ้นเมื่อสิ้นสุดการทดลองใช้

เมื่อสิ้นสุดการทดลองใช้ Support ข้อมูลและการตั้งค้าของคุณยังคงอยู่ดังเดิม คุณสามารถเข้าสู่ระบบและเลือกซื้อแผนได้

ซื้อ

ฉันจะซื้อซอฟต์แวร์ได้อย่างไร

หลังจากคลิก “ซื้อเลย” คุณจะสร้างบัญชี Zendesk และสามารถซื้อซอฟต์แวร์ได้ทันที คุณสามารถชำระเงินด้วยบัตรเครดิต หรือเลือกให้ส่งใบแจ้งหนี้สำหรับแผนระดับสูงสุด

สัญญามีระยะเวลาเท่าใด ฉันสามารถเปลี่ยนแผนได้หรือไม่

แผนเป็นแบบเดือนต่อเดือนหรือรายปี คุณสามารถเริ่ม/หยุดหรือเปลี่ยนแปลงแผนของคุณได้ทุกเวลา (แต่โปรดทราบว่าไม่มีการคืนเงินสำหรับกรณีการยกเลิกหรือการดาวน์เกรด) ติดต่อเราเพื่อขอข้อมูลเพิ่มเติม

ฉันสามารถแยกซื้อคุณสมบัติต่างๆ ได้ไหม มีค่าใช้จ่ายเพิ่มเติมหรือไม่

ด้วย Add-ons สำหรับผลิตภัณฑ์ Support ของเรา คุณสามารถเลือกคุณสมบัติเพิ่มเติมที่คุณต้องการให้มีนอกเหนือจากแผนหลักของคุณได้ ติดต่อเรา หากคณต้องการซื้อ Add-on

มีราคาเหมารวมสำหรับ Support, Chat, Guide และ Talk หรือไม่

แผนระดับ Lite ของ Chat และ Guide มาพร้อมกับ Support ฟรี คุณสามารถอัพเกรดเป็นแผน Chat แบบ Basic, Advanced หรือ Premium จากแผนใดก็ตามของ Support คุณสามารถอัพเกรดเป็นแผน Guide Professional จากแผนใดก็ตามของ Support คุณสามารถเพิ่ม Talk ให้กับ Support แผนใดก็ได้

มีส่วนลดใดให้เลือกบ้าง

เราเสนอราคาส่วนลดสำหรับการสมัครใช้บริการเรียกเก็บรายปี ราคาที่แสดงในหน้านี้เป็นราคาลดรายปี

ทั่วไป

เจ้าหน้าที่คืออะไร

เจ้าหน้าที่คือคำที่ใช้เรียกใครก็ตามที่ทำการเข้าสู่ผลิตภัณฑ์ Zendesk ตัวแทนการ ผู้ดูแลระบบ หรือผู้จัดการต้องใช้สิทธิ์การใช้งานสำหรับเจ้าหน้าที่ของตนเอง เพิ่มเจ้าหน้าที่ได้มากเท่าที่คุณต้องการไม่ว่าจะในแผนใด

ผลิตภัณฑ์ Zendesk สามารถทำงานร่วมกันกับแอพพลิเคชันอื่นๆ ได้หรือไม่

เรามีแอพและการทำงานร่วมกันนับร้อยที่สร้างไว้แล้วล่วงหน้า แต่คุณสามารถสร้างแทบทุกอย่างที่คุณต้องการบนแพลตฟอร์ม Zendesk ผ่านการใช้ API สำหรับนักพัฒนา

ขอมูลของฉันจะคงความเป็นส่วนตัวและปลอดภัยหรือไม่

ปลอดภัย เราได้รวมคุณสมบัติการรักษาความปลอดภัยระดับ Enterprise พร้อมการตรวจสอบที่ครอบคลุมทั้งแอพพลิเคชัน ระบบ และเครือข่ายเพื่อให้ข้อมูลของลูกค้าและองค์กรได้รับการปกป้องตลอดเวลา ลองดูมาตรการการรักษาความปลอดภัยต่างๆ ที่เราใช้เพื่อปกป้องธุรกิจและลูกค้าของคุณ

หากยังมีคำถามเพิ่มเติม

เรามี ศูนย์ความช่วยเหลือ ที่เปิด 24/7 หรือติดต่อไปที่ทีมสนับสนุนสากลของเราได้ เราคอยให้ความช่วยเหลืออยู่
