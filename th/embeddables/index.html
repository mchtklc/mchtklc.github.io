Embeddables

Embeddables

ให้ลูกค้าของคุณได้รับการสนับสนุนลูกค้าโดยตรง

ลองใช้ดู

มีทุกอย่างที่ลูกค้าต้องการในทันทีที่ต้องใช้

Embeddables ของ Zendesk ช่วยให้ธุรกิจของคุณสามารถหยิบยื่นความช่วยเหลือให้แก่ลูกค้าได้ทุกที่ทุกเวลาที่ลูกค้าต้องการ ด้วย Embeddables คุณสามารถติดต่อไปยังลูกค้า เสนอความช่วยเหลือ และเริ่มต้นสนทนากับลูกค้าได้ด้วยตัวเอง และเพราะมันผสานรวมเข้ากับแอพสำหรับอุปกรณ์พกพาหรือหน้าเว็บ ประสบการณ์ที่ได้ก็จะยิ่งวิเศษ

  • ศูนย์ความช่วยเหลือ

  • แชท

  • ใบสั่งงาน

Mobile SDK

Web Widget

ทำงานอย่างไร

การบริการลูกค้าไม่มีสะดุด
ราวใช้เวทมนตร์

Zendesk Embeddables คือการผสมผสานกันของ Web Widget และ Mobile SDK สำหรับทั้ง Android และ iOS ซึ่งจะทำให้นักพัฒนาสามารถฝังฟังก์ชันต่างๆ ของ Zendesk ทั้ง ศูนย์ความช่วยเหลือ การแชท หรือใบสั่งงาน ภายในแอพ เว็บไซต์ หรืออุปกรณ์แบบสแตนด์อโลน โดยจะช่วยรักษาความสม่ำเสมอของแบรนด์ เพิ่มประสิทธิภาพของการโต้ตอบการสนับสนุน และจำกัดการต่อต้านจากผู้ใช้ให้น้อยที่สุด

สิ่งที่ทำให้ Embeddables นั้นพิเศษ

  • ประสบการณ์ของลูกค้าที่ดีขึ้น

    การสนับสนุนไม่ควรจะอยู่ชั้นต่ำสุดของประสบการณ์ของลูกค้า การฝังการสนับสนุนในที่ที่ลูกค้าของคุณอยู่ ทั้งในทุกช่องทาง อุปกรณ์ และสภาพแวดล้อม ช่วยลดความจำเป็นในการขอความช่วยเหลือและทำให้การบริการลูกค้าที่ดีเป็นส่วนหนึ่งของผลิตภัณฑ์

  • แก้ปัญหาได้เร็วขึ้น เมื่อมีข้อมูลข้างเคียง

    ลูกค้าไม่ควรจะต้องรับบทนักสืบด้วยตัวเอง เมื่อพูดถึงปัญหาทางเทคนิค Zendesk Embeddables จะสร้างข้อมูลเกี่ยวกับอุปกรณ์หรือเวอร์ชันของแอพที่กำลังใช้อยู่โดยอัตโนมัติเมือเกิดปัญหาขึ้น ซึ่งทำให้คุณรู้ล่วงหน้าและให้การสนับสนุนได้อย่างรวดเร็ว

  • เปิดโอกาสให้คุณ
    ได้เติบโต

    Embeddables ช่วยลดภาระที่ไม่จำเป็นด้วยการให้บริการข้อมูลที่ถูกต้อง ในเวลาที่ใช่ ในสถานที่ที่เหมาะสม การผสมผสานกันอย่างราบรื่นของการบริการลูกค้าและเวิร์กโฟลว์ที่คล่องตัวนี้จะทำให้การสนับสนุนมีประสิทธิภาพมากขึ้นและลดการใช้งานทรัพยากรที่จำเป็นต่อการเติบโตของธุรกิจ

Mobile SDK

ให้การสนับสนุนได้จากแอพสำหรับอุปกรณ์พกพาของคุณ

ยกระดับการสนับสนุนทางอุปกรณ์พกพาของคุณด้วยการบริการตนเอง

ช่วยให้ลูกค้าค้นหาคำตอบด้วยตนเองในศูนย์ความช่วยเหลือที่สามารถเข้าถึงได้จากอุปกรณ์พกพา ทำให้ได้รับคำตอบเร็วขึ้น และปริมาณการสนับสนุนลดลง

ใช้งานทางแอพได้ทันท่วงที

สำหรับข้อสงสัยที่ต้องการความช่วยเหลือ ก็สามารถทำการติดต่อหรือมีส่วนร่วมในแบบเรียลไทม์ด้วยการแชทสด ประสบการณ์ที่ราบรื่นโดยที่ผู้ใช้ไม่ต้องออกจากแอพของคุณ

มีบริบทมากขึ้นเพื่อให้แก้ปัญหาได้เร็วขึ้น

จับข้อมูลสำคัญเกี่ยวกับลูกค้า อย่างอุปกรณ์หรือระบบปฏิบัติการโดยอัตโนมัติเพื่อให้เจ้าหน้าที่ฝ่ายสนับสนุนทราบบริบทที่จำเป็น

Web Widget

มีส่วนร่วมกับผู้ใช้บนเว็บไซต์ของคุณ

ฝังการสนับสนุนในแบบของคุณ

เลือกวิธีการสนับสนุนบนเว็บไซต์ของคุณที่ทำงานร่วมกับ Web Widget ซึ่งสามารถผสานรวมการค้นหาจากฐานความรู้ แบบฟอร์มการสนับสนุน แชทสด หรือขอให้โทรกลับ

คำแนะนำที่ชาญฉลาดเพื่อช่วยคุณประหยัดเวลา

แทนที่จะบังคับให้ลูกค้าของคุณต้องตามหาหรือออกจากเว็บไซต์เพื่อขอความช่วยเหลือ เรามี Web Widget ที่จะแนะนำบทความจากฐานความรู้ในยามที่พวกเขาต้องการ

ประสบการณ์สนับสนุนที่ราบรื่น

Web Widget นั้นได้รับการออกแบบให้เหมาะสมกับการมอบประสบการณ์ผู้ใช้ที่เรียบง่าย ไม่ว่าลูกค้าจะใช้งานเครื่องเดสก์ท็อป หรืออุปกรณ์พกพา นอกจากนี้ยังแปลเป็นภาษาของลูกค้าตามการตั้งค่าของพวกเขา

  • ใช้งานวิดเจ็ต

สำหรับนักพัฒนา

สิ่งที่ต้องการในการฝัง Web Widget ลงบนเว็บไซต์ของคุณก็มีแค่ Javascript เล็กน้อยเท่านั้น เมื่อใช้งาน Web Widget API คุณก็จะสามารถส่งผ่านข้อมูลของคุณเอง ปรับแต่งรูปลักษณ์ และสามารถควบคุมการทำงานวิดเจ็ตได้มากขึ้น

Zendesk API

หากคุณสร้างไว้ ก็ใช้ได้

Mobile SDK และ Web Widget เป็นเครื่องมือที่มีประสิทธิภาพ แต่นักพัฒนาก็ยังต้องการคุณสมบัติอื่นเพิ่มเติม นั่นจึงเป็นเหตุผลที่ Zendesk เสนอบริการ REST API ทำให้การปรับแต่งสามารถทำได้อย่างอิสระและล้ำสมัยกว่าที่เคย API ของเรามีเกินกว่า 400 endpoints ทำให้คุณสามารถสร้างสรรค์ได้ตั้งแต่เริ่มได้ในทันที นี่คือประสบการณ์การสนับสนุนตั้งแต่เริ่มต้นที่คุณต้องการในทุกอุปกรณ์
