ลงลึกในการใช้ Zendesk Support

ต้องการเคล็ดลับการเริ่มทดลองใช้งาน Support อยู่หรือ ลองดูที่ การสอนการเริ่มต้นใช้งาน

ชมวิดีโอเพิ่มเติมเกี่ยวกับคุณสมบัติอื่นๆ ของ Zendesk Support:

  • กลุ่ม
  • บทบาทและการให้อนุญาต
  • แบบสำรวจ CSAT
  • การตั้งเวลาทำการ
แหล่งข้อมูลอื่นๆ ที่จะช่วยคุณได้:

ดูแนวทางการใช้ Zendesk Support ที่ดีที่สุดได้ ที่นี่

หากต้องการความช่วยเหลือแบบ 1:1 เกี่ยวกับการสาธิต ส่งอีเมลไปที่ demo@zendesk.com

คำถามที่พบบ่อย

ใบสั่งงานใน Zendesk Support ทำงานอย่างไร

การจัดการใบสั่งงานใน Zendesk Support นั้นง่ายและเป็นธรรมชาติ สำหรับเรื่องพื้นฐานของใบสั่งงาน คลิก ที่นี่

ฉันจะตั้งค่าเริ่มใช้ทริกเกอร์และงานอัตโนมัติได้อย่างไร

คุณสมบัติด้านประสิทธิภาพของ Zendesk Support นั้นตั้งค่าเริ่มใช้ได้ง่ายช่วยให้เจ้าหน้าที่เป็นอิสระจากงานที่ต้องทำซ้ำๆ คลิกที่นี่ เพื่ออ่านเพิ่มเติมเกี่ยวกับทริกเกอร์และงานอัตโนมัติ

แบบฟอร์มใบสั่งงานใน Zendesk Support ทำงานอย่างไร

ลองดู วิดีโอนี้ เพื่อให้เห็นภาพรวมหรืออ่าน บทความนี้ เพื่อทราบเพิ่มเติมเกี่ยวกับวิธีการจัดการแบบฟอร์มใบสั่งงาน

Zendesk Support นั้นปรับแต่งได้มากน้อยแค่ไหน

ระบบของ Zendesk สามารถปรับแต่งได้มากมาย องค์กรที่สนับสนุนผู้ประกอบการทั้งเล็กและใหญ่ต่างก็เลือกใช้ เพราะสามารถปรับขนาดได้ตั้งแต่เจ้าหน้าที่จำนวนหยิบมือเดียวไปจนถึงองค์กรที่มีเจ้าหน้าที่ระดับหลายหมื่นคน

แล้วจะตั้งค่าหลายศูนย์ความช่วยเหลืออย่างไร

คุณสามารถจัดให้มีศูนย์ความช่วยเหลือหลายแห่งด้วยการใช้คุณสมบัติหลายแบรนด์ อ่านเพิ่มเติม ที่นี่

Zendesk ทำงานร่วมกับแอพพลิเคชันหรือบริการใดได้บ้าง

Zendesk ทำงานร่วมกับระบบและเครื่องมืออื่นๆ นับร้อยรายการ ซึ่งสามารถดูรายนามแอพและการทำงานร่วมกันทั้งหมดได้ ที่นี่

Zendesk API ทำงานอย่างไร

Zendesk มี API กว่า 100+ ให้คุณได้ผสานการใช้งาน อ่านเพิ่มเติมในแหล่งข้อมูล API ต่างๆ ของเราได้ ที่นี่

แผนแต่ละแผนต่างกันอย่างไร และจะรู้ได้อย่างไรว่าแผนไหนจะเหมาะกับฉัน

สามารถตรวจสอบข้อมูลของแผนต่างๆ ที่หน้าเปรียบเทียบแผนของเราได้ ที่นี่ หากคุณต้องการหารือเรื่องแผนต่างๆ เพียงส่งอีเมลมาที่ demo@zendesk.com แล้วเราจะติดต่อกลับไป

Zendesk มีการปฏิบัติตามและการรักษาความปลอดภัยอย่างไร

Zendesk จริงจังกับการรักษาปลอดภัยของลูกค้าอย่างยิ่ง เราได้รวมคุณสมบัติการรักษาความปลอดภัยระดับ Enterprise พร้อมการตรวจสอบที่ครอบคลุมทั้งแอพพลิเคชัน ระบบ และเครือข่ายเพื่อให้ข้อมูลของลูกค้าและองค์กรได้รับการปกป้องตลอดเวลา คลิกที่นี่ เพื่ออ่านเพิ่มเติม
