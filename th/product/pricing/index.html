                                                  ฉันสามารถแยกซื้อคุณสมบัติต่างๆ ได้ไหม มีค่าใช้จ่ายเพิ่มเติมหรือไม่

                                                  ด้วย Add-ons สำหรับผลิตภัณฑ์ Support ของเรา คุณสามารถเลือกคุณสมบัติเพิ่มเติมที่คุณต้องการให้มีนอกเหนือจากแผนหลักของคุณได้ ติดต่อเรา หากคณต้องการซื้อ Add-on

                                                  มีราคาเหมารวมสำหรับ Support, Guide, Chat และ Talk หรือไม่

                                                  ใช่แล้ว Support Suite ได้รวมเอาผลิตภัณฑ์ทั้งสี่ให้กับเจ้าหน้าที่ทุกคนของคุณโดยเริ่มที่ $89 ต่อเจ้าหน้าที่ต่อเดือน อ่านรายละเอียดเกี่ยวกับ Support Suite ที่นี่ แต่ละที่นั่งใช้กับเจ้าหน้าที่คนเดียวกันในทุกผลิตภัณฑ์ที่มากับ Support Suite

                                                  เมื่อใช้ Support Suite คุณจะไม่สามารถเลือกเพิ่มเจ้าหน้าที่เฉพาะ Support หรือ Chat เท่านั้น โดยคุณสามารถเพิ่มจำนวนเจ้าหน้าที่ใน Support Suite ได้ ซึ่งจะมาพร้อมกับที่นั่งสำหรับ Support, Guide, Chat และ Talk

                                                  Sell

                                                  Zendesk มีซซอฟต์แวร์การขายอัตโนมัติ (SFA) ให้บริการไหม

                                                  Sell และ Support ทำงานร่วมกันอย่างไร

                                                  Sell และ Support นั้นมีการผสานรวมกัน ช่วยให้พนักงานขายสามารถดูข้อมูลใบสั่งงานที่อยู่ถัดจากโอกาสในการขายได้ ฝ่ายสนับสนุนสามารถดูกิจกรรมการขายในประวัติการโต้ตอบของลูกค้าได้ใน Support นอกจากนี้ยังสามารถดูข้อมูลเกี่ยวกับข้อเสนอพิเศษที่มีในแอพ Sell ใหม่ล่าสุดสำหรับ Support และยังแจ้งยอดขายได้ หากบทสนทนาเพื่อการสนับสนุนกลายเป็นโอกาสในการขายได้

                                                  แพลตฟอร์ม

                                                  บันทึก Custom Object คืออะไร

                                                  Custom Object API ช่วยให้คุณสามารถกำหนดประเภท Object ใหม่ใน Zendesk จากนั้นสร้าง Object จากประเภท Object ใหม่ ซึ่งบันทึก Custom Object ก็คืออินสแตนซ์ของประเภท Object ที่กำหนดไว้ ตัวอย่างเช่น หากคุณมีธุรกิจที่พักให้เช่า “อสังหาริมทรัพย์ให้เช่า” ก็จะเป็นประเภท Custom Object และอสังหาริมทรัพย์ที่ให้เช่าแต่ละแห่งที่เกี่ยวข้องกับผู้เช่าจะแสดงด้วยบันทึก Custom Object หนึ่งรายการ อ่านรายละเอียดที่นี่.

                                                  Zendesk Evens คืออะไร

                                                  Zendesk Events จะปรากฏในประวัติการโต้ตอบ ทำให้เห็นกิจกรรมจากผลิตภัณฑ์ของ Zendesk เช่น Support หรือ Guide โดยกิจกรรมบางส่วนมีอยู่ใน EAP แล้ว ซึ่งแผน Sunshine ทุกประเภทจะมีสิทธิ์การเข้าถึงคุณสมบัตินี้อยู่ด้วย อ่านรายละเอียดที่นี่.

                                                  Custom Event คืออะไร

                                                  Sunshine Event API ช่วยให้คุณสามารถสร้างไทม์ไลน์ของการปฏิสัมพันธ์กับลูกค้าทั้งหมดจากแหล่งภายนอกทุกๆ แหล่ง โดย Custom Event อาจเป็นการปฏิสัมพันธ์กับลูกค้าที่บันทึกไว้ในรูปแบบใดก็ได้ จากระบบภายนอก เช่น ธุรกรรมการสั่งซื้อ หรือการเข้าชมเว็บไซต์ ซึ่งการเข้าถึง Custom Events (กิจกรรมนอก Zendesk) ต้องใช้แผนประเภท Sunshine Pro ขึ้นไป อ่านรายละเอียดที่นี่.

                                                  โปรไฟล์ภายนอกคืออะไร

                                                  Sunshine Profile API ช่วยให้คุณสามารถจัดการข้อมูลลูกค้าในระบบ แหล่งข้อมูล และช่วงเวลาต่างๆ ได้ ซึ่งโปรไฟล์ภายนอกประกอบด้วยตัวระบุอย่างน้อยหนึ่งรายการสำหรับบุคคลในระบบนอก Zendesk อย่างน้อยหนึ่งระบบ รวมถึงแอตทริบิวต์แบบกำหนดเองที่มีให้เลือกใช้ได้กับบุคคลดังกล่าว อ่านรายละเอียดที่นี่

                                                  ข้อจำกัด Sunshine Lite, Pro และ Enterprise API หมายถึงอะไร

                                                  Lite (คำร้องขอต่อนาที): 500 สำหรับ Custom Objects (Custom Events, Unified Profiles และ AWS Events Connector ไม่รวมอยู่ในแผน Lite)

                                                  Pro (คำร้องขอต่อนาที): 750 สำหรับ Custom Objects, 250 สำหรับ Custom Events, 250 สำหรับ Unified Profiles และ 700 สำหรับ AWS Events Connector

                                                  Enterprise (คำร้องขอต่อนาที): 1,000 สำหรับ Custom Objects, 500 สำหรับ Custom Events, 500 สำหรับ Unified Profiles และ 700 สำหรับ AWS Events Connector

                                                  AWS Events Connector คืออะไร

                                                  AWS Events Connector จะช่วยให้คุณสามารถสตรีมกิจกรรม Zendesk ในแบบเรียลไทม์ลงในบริการ AWS หลักๆ ได้ อ่านรายละเอียดเกี่ยวกับ Connector ที่นี่
